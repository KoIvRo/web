# Блог на FastAPI и React

Блог с системой аутентификации, созданный на FastAPI (бэкенд) и React (фронтенд).

## Функциональность

### Бэкенд (FastAPI)
- **Аутентификация**: JWT токены с access/refresh механизмом
- **Посты**: Создание, редактирование, удаление, просмотр постов
- **Комментарии**: Добавлние комментариев под постами
- **Категории**: Фильтрация постов по категориям
- **Пользователи**: Регистрация, вход и выход из аккаунта

### Фронтенд (React)
- **Главная страница**: Информация о авторе, последние посты
- **Блог**: Просмотр всех постов с фильтрацией по категориям
- **Детали поста**: Полный текст поста и система комментариев
- **Аутентификация**: Вход и регистрация
- **Управление контентом**: Создание и редактирование постов/комментариев

## Технологии

### Бэкенд
- **FastAPI** - REST API Python фреймворк
- **SQLAlchemy** - ORM для работы с базой данных
- **JWT** - аутентификация через токены
- **Passlib** - хеширование паролей
- **SQLite** - база данных

### Фронтенд
- **React** - пользовательский интерфейс
- **React Router** - навигация между страницами
- **Axios** - HTTP клиент для работы с API
- **Context API** - управление состоянием аутентификации

## Установка и запуск

### Предварительные требования
- Python 3.8+
- Node.js 14+
- pip (менеджер пакетов Python)
- npm (менеджер пакетов Node.js)

### Установка бэкенда

```bash
git clone <url-репозитория>
cd api
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload --port 8000
```
Бэкенд будет доступен по адресу: http://localhost:8000
Документация API: http://localhost:8000/docs

### Установка фронтенда
```bash
cd ../my-front
npm install
npm start
```
Фронтенд будет доступен по адресу: http://localhost:3000
